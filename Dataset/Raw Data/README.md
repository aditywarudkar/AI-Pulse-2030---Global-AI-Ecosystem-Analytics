# 📂 Raw Data

This folder contains the original, unmodified datasets used in the **AI Pulse 2030 – Global AI Ecosystem Analysis** project.

## Contents

- Original source datasets
- Data collected before any cleaning or transformation
- Files used as the starting point for data preparation

## Notes

- No modifications have been made to these files.
- These datasets serve as the source for the cleaned datasets stored in the **02_Clean** folder.
- The raw data is preserved to ensure data integrity and reproducibility.
