# 🗄️ SQL

This folder contains all SQL scripts used to analyze the **AI Pulse 2030 – Global AI Ecosystem Analysis** dataset.

## Contents

- Database creation scripts
- Data import queries
- Data cleaning queries
- Exploratory Data Analysis (EDA)
- Business analysis queries
- KPI calculations
- Advanced SQL queries

## SQL Concepts Used

- SELECT Statements
- WHERE, ORDER BY
- GROUP BY & HAVING
- Aggregate Functions
- CASE Statements
- JOINS (INNER, LEFT, RIGHT)
- Common Table Expressions (CTEs)
- Subqueries
- Window Functions
- Views
- Date & String Functions

## Purpose

These SQL scripts transform raw data into meaningful business insights by performing data exploration, aggregation, and analytical queries.

## Output

The results generated from these queries support the Power BI dashboard, KPI reporting, and business decision-making presented in this project.
